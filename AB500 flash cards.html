Allen-Bradley SLC 500 Flash Cards

1) What software is commonly used to program Allen-Bradley SLC 500 PLCs?
A) RSLogix 500
B) Studio 5000
C) Connected Components Workbench
D) TIA Portal
✅ Answer: A) RSLogix 500
2) Which SLC 500 CPU supports Ethernet communications natively?
A) SLC 5/03
B) SLC 5/04
C) SLC 5/05
D) SLC 5/01
✅ Answer: C) SLC 5/05
3) What communication port type is standard on SLC 5/04?
A) DH-485
B) DF1
C) Ethernet
D) DH+
✅ Answer: D) DH+
4) What is the maximum number of I/O modules in an SLC 500 rack?
A) 5
B) 7
C) 10
D) 30
✅ Answer: C) 10
5) Which instruction in RSLogix 500 is used to move data from one location to another?
A) ADD
B) MOV
C) TON
D) CPT
✅ Answer: B) MOV
6) In RSLogix 500, what file number is usually the Data File for timers?
A) B3
B) T4
C) N7
D) F8
✅ Answer: B) T4
7) What does DH-485 stand for?
A) Device Highway 485
B) Data Highway 485
C) Distributed Highway 485
D) Device Hub 485
✅ Answer: B) Data Highway 485
8) What is the purpose of the SLC 500 “Keyswitch”?
A) Enables memory expansion
B) Selects program/run/test modes
C) Controls power to I/O modules
D) Changes IP address
✅ Answer: B) Selects program/run/test modes
9) What is the typical voltage for SLC 500 discrete inputs?
A) 3VDC
B) 24VDC or 120VAC
C) 480VAC
D) 12VDC only
✅ Answer: B) 24VDC or 120VAC
10) What is the RS-232 port typically labeled on an SLC 5/03 CPU?
A) Channel 0
B) Channel 1
C) Ethernet Port
D) DH+ Port
✅ Answer: A) Channel 0
